Preprocessor Directives

Preprocessor directives begin with # and are processed before compilation. Common ones include #include for including libraries and #define for constants or macros.

CODE:


// Program: Using #define
#include <stdio.h>
#define PI 3.14

int main() {
    float radius = 5.0;
    float area = PI * radius * radius;
    printf("Area: %.2f\n", area);
    return 0;
}

Loops in C

Loops allow repeated execution of code blocks. C supports for, while, and do-while.

CODE:


// Program: Print numbers 1 to 5 using for loop
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Formatted I/O Functions

Formatted input/output functions like scanf and printf allow you to take user input and display output with format specifiers.

CODE:


// Program: Formatted I/O
#include <stdio.h>

int main() {
    int age;
    float height;
    printf("Enter your age and height: ");
    scanf("%d %f", &age, &height);
    printf("Age: %d, Height: %.2f\n", age, height);
    return 0;
}

Control Flow

Control flow in C determines the order in which statements are executed. It includes if-else, loops, and switch-case structures.

CODE:


// Program: Simple if-else example
#include <stdio.h>

int main() {
    int num = 10;
    if (num > 0)
        printf("Positive number\n");
    else
        printf("Non-positive number\n");
    return 0;
}

Pattern Printing

Patterns are printed using nested loops. Common examples include triangles, squares, and pyramids of characters or numbers.

CODE:


// Program: Right-angled triangle pattern
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        for (int j = 1; j <= i; j++) {
            printf("* ");
        }
        printf("\n");
    }
    return 0;
}

String Handling Functions

C provides functions like strlen, strcpy, and strcmp to handle strings.

CODE:


// Program: Using strlen and strcpy
#include <stdio.h>
#include <string.h>

int main() {
    char str1[] = "Hello";
    char str2[20];
    strcpy(str2, str1);
    printf("Copied string: %s\n", str2);
    printf("Length: %lu\n", strlen(str2));
    return 0;
}

Tokens in C

Tokens are the smallest elements in a C program. These include: keywords, identifiers, constants, operators, and special symbols.

CODE:


// Example: Tokens
#include <stdio.h>

int main() {

}

Arrays

Arrays store multiple values of the same type. Access elements using their index (starting from 0).

CODE:


// Program: Print array elements
#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    return 0;
}

Functions

Functions help break code into reusable blocks. You define them once and call them multiple times.

CODE:


// Program: Simple function

void greet() {
    printf("Welcome to C programming!\n");
}

int main() {

    greet();
    return 0;
}

Recursion

A recursive function is one that calls itself. Useful for problems like factorial, Fibonacci, and tree traversal.

CODE:


// Program: Factorial using recursion
#include <stdio.h>

int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}
