Preprocessor Directives

Preprocessor directives in C are commands that begin with # and are processed before compilation...

CODE:


// Program 11: Preprocessor Directives
#include <stdio.h>
#define DEBUG 1
int main() {
    #ifdef DEBUG
        printf("Debug mode is enabled.\\n");
    #endif
    return 0;
}
